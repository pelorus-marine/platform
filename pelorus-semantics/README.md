# pelorus-semantics

Small, dependency-free definitions for **shared signal identity** across **Pelorus Core** (CAN / DCID) and **Pelorus Stream** (Ethernet telemetry, media paths).

Normative semantics stay in **`specifications/`** (catalog, `Vessel.*`). This crate only carries types that transports and higher layers agree on — no DBC parsers, no Stream codecs, no fusion policy.

Workspace root: `../README.md`.
